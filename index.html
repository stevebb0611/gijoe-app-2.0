DESIGN HANDOFF · LOCAL DEV BUNDLE

G.I. JOE COLLECTION // TRACKER

A personal inventory & condition tracker for a 700–800 figure vintage G.I. Joe collection. This bundle is the design source of truth: high-fidelity reference mockups for screen-by-screen review, plus the spec docs that define the data model, decisions, and standards. Serve it locally (below), open any screen, then hand it to Claude Code to build from.

RUN IT LOCALLY ~60 SECONDS

The prototypes compile JSX in the browser, so they must be served over http://localhost — opening the files directly (file://) will leave the screens blank. Any static server works; here's the zero-install path.

Open the folder in VS Code. File → Open Folder → choose this design_handoff_inventory folder. Open the built-in terminal (Ctrl/Cmd + `).
Start a static server from the terminal (requires Node, which ships with VS Code workflows): # zero-install, recommended npx serve . # …or with the package.json in this folder npm start
Open the printed URL (usually http://localhost:3000) in your browser. This index page loads first — open any reference screen below (start with Inventory, the design of record).
Hand it to Claude Code. Run claude in this same folder and start with: "Read CLAUDE.md, then the README and spec docs." It now has the full design context plus a live preview to compare against.

No Node? Use python3 -m http.server 3000 instead, or install the Live Server VS Code extension and click "Go Live". Heads-up: screens pull React, Babel & fonts from CDNs, so keep an internet connection for the first load.
